API Endpoint for Sensor Data

This endpoint allows automated systems, such as **IoT sensors and drones**, to securely send data to our platform.

POST /api/v1/sensor-data


{
"device_id": "sensor_001",
"timestamp": "2025-08-18T08:00:00Z",
"location": {
"latitude": 34.052235,
"longitude": -118.243683
},
"data": {
"temperature": 25.5,
"humidity": 60,
"air_quality": "good"
},
"image_url": "https://example.com/drone_image_123.jpg"
}

Data can be sent as JSON or CSV packets. Drone images should be uploaded to a CDN and the URL included in the payload.
